# GabaHR Test

Тестовое задание — список пользователей с поиском и пагинацией на базе [DummyJSON](https://dummyjson.com).

## 🚀 Запуск

```bash
npm install
npm run dev
```

## 🛠 Стек

- **React 19** + **TypeScript**
- **Vite** — сборка
- **TanStack Query v5** — запросы и кеширование
- **React Router v7** — роутинг
- **SCSS Modules** — стили
- **ESLint + Prettier** — линтинг и форматирование

## 📦 Скрипты

| Команда           | Описание        |
| ----------------- | --------------- |
| `npm run dev`     | Дев-сервер      |
| `npm run build`   | Продакшн-сборка |
| `npm run preview` | Превью сборки   |
| `npm run lint`    | Запуск ESLint   |
